# NPM Packages

- [validator](https://www.npmjs.com/package/validator)
- [chalk](https://www.npmjs.com/package/chalk)

### Installation

`npm i chalk`

`npm i validator`

### Verify Installation

You can verify installation by checking it in package.json in root of your project

```
  "dependencies": {
    "chalk": "^2.4.2",
    "validator": "^11.0.0"
  }
```
